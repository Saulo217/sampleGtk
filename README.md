# Sample GUI with Gtk 3

## Linux Systems

requirements:
	gcc 10.2 or later
	make 4.3 or later
	libgtk-3-0
	libgtk-3-dev
	libopengl0
	libopengl-dev

run instructions:
> type "make" or "make run" on your terminal for compile and run the application

clean instructions:
> type "make clean" on your terminal for delete the executable binary
